Os aceleradores do Tableau são painéis pré-criados projetados para ajudar você a obter um impulso inicial na análise de dados.

Adaptados a setores específicos e aplicativos corporativos, os Aceleradores são criados com dados de amostra que você pode trocar por seus próprios dados,

permitindo que você descubra insights com configuração mínima.

Com o maior repositório de visualizações de dados do mundo para aprender, o Tableau Public facilita o desenvolvimento de habilidades de dados.
